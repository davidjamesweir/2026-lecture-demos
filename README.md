# Running the demos on mybinder.org

This is the recommended approach, at least in the lectures - unless you are quick at cloning this repository and (if necessary) setting up a virtual environment.

Clicking the links below will launch an interactive Python notebook on the [mybinder.org service](https://mybinder.org/). Note that **your changes will not be saved** if you modify the notebooks on mybinder.org.

Also, please be patient, it can take a little while for the notebooks to start up.
